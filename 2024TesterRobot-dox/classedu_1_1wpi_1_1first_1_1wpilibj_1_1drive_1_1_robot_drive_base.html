◆ setDeadband()

void edu.wpi.first.wpilibj.drive.RobotDriveBase.setDeadband ( double deadband)
inline

Sets the deadband applied to the drive inputs (e.g., joystick values).

The default value is {@value #kDefaultDeadband}. Inputs smaller than the deadband are set to 0.0 while inputs larger than the deadband are scaled from 0.0 to 1.0. See edu.wpi.first.math.MathUtil#applyDeadband.

Parameters
deadbandThe deadband to set.
