static double edu.wpi.first.wpilibj.simulation.BatterySim.calculateDefaultBatteryLoadedVoltage ( double... currents)
inlinestatic

Calculate the loaded battery voltage. Use this with RoboRioSim#setVInVoltage(double) to set the simulated battery voltage, which can then be retrieved with the edu.wpi.first.wpilibj.RobotController#getBatteryVoltage() method. This function assumes a nominal voltage of 12v and a resistance of 20 milliohms (0.020 ohms)

Parameters
currentsThe currents drawn from the battery.
Returns
The battery's voltage under load.

Parameters
