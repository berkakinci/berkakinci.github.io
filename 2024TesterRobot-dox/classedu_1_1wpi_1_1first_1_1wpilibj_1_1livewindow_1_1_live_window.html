◆ setEnabled()

static synchronized void edu.wpi.first.wpilibj.livewindow.LiveWindow.setEnabled ( boolean enabled)
inlinestatic

Set the enabled state of LiveWindow.

If it's being enabled, turn off the scheduler and remove all the commands from the queue and enable all the components registered for LiveWindow. If it's being disabled, stop all the registered components and re-enable the scheduler.

TODO: add code to disable PID loops when enabling LiveWindow. The commands should re-enable the PID loops themselves when they get rescheduled. This prevents arms from starting to move around, etc. after a period of adjusting them in LiveWindow mode.

Parameters
enabledTrue to enable LiveWindow.
